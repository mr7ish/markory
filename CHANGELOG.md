# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this file is intended to help future releases stay readable and easy to maintain.

## [Unreleased]

### Added

- Built the main bookmark workspace based on `WXT + Vue 3 + TypeScript`
- Added bookmark browsing with breadcrumb navigation
- Added global bookmark search with directory context
- Added bookmark and folder creation, editing and deletion
- Added `Focus` view for frequently used bookmarks
- Added `Group` view for saving current window tabs as bookmark groups
- Added the ability to reopen grouped bookmarks as browser tab groups
- Added recycle bin workflow with restore, clear and delayed expiration cleanup
- Added right-click context menu actions for common bookmark operations
- Added drag-and-drop moving and tree-based move dialog
- Added bookmark favicon fallback and optional floating preview
- Added dark / light theme switching
- Added Chinese and English language switching
- Added browser context menu entry for grouping current window tabs
- Added real-time syncing for bookmark create, update, remove and move events

### Documentation

- Rewrote `README.md` into a product-style project homepage
- Added installation, build, publishing and preview guidance
- Added screenshot placeholder guidance for future documentation assets

## [0.1.0] - 2026-04-03

### Added

- Initial public-ready documentation baseline
- Current core feature set for bookmark management, grouping and recycle workflow
