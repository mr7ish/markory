# Docs Assets

This directory is reserved for project documentation assets.

Recommended contents:

- Product screenshots for `README.md`
- Animated GIF demonstrations
- Release notes related images
- Store listing materials for Chrome Web Store or Firefox Add-ons

Suggested file names:

- `home.png`
- `search.png`
- `group.png`
- `recycle.png`
- `settings.png`

If you later add screenshots, the preview section in the root `README.md` can point directly to files in this directory.
